# Healthcare Data Analysis System

Academic SQL project focused on healthcare data integration and analysis using information collected from multiple hospitals and datasets.

The project involved importing and organizing healthcare-related data from multiple Excel files into a relational database system, followed by the development of analytical SQL queries and database logic features.

---

# Project Overview

The main objective of the project was to analyze hospital-related information and extract meaningful insights through SQL queries and database mechanisms.

Examples of analysis performed include:
- Hospitals with the longest waiting times
- Hospitals with the best ratings in specific categories
- Number of hospitals by geographic area
- Comparative healthcare statistics between regions
- Hospital performance analysis

---

# Database Features

The project also explored advanced database concepts such as:
- Stored Procedures
- Functions
- Triggers

## Example Trigger Logic

A trigger was implemented to automatically validate hospital rating values when new hospital data was inserted into the database.

Based on the rating score, the system automatically generated qualitative comments such as:
- "Good"
- "Very Good"
- "Excellent"

This allowed automatic classification and validation of inserted data.
